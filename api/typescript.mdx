## Installation

Install the Membrane TypeScript client from npm:

```bash
npm install @gustycube/membrane
```

The client requires Node.js 20 or higher.

## Quick Start

```typescript
import { MembraneClient, Sensitivity } from '@gustycube/membrane';

const client = new MembraneClient('localhost:9090');

// Ingest an event
const record = await client.ingestEvent(
  'file_edit',
  'src/main.ts',
  { 
    summary: 'Updated main entry point',
    sensitivity: Sensitivity.LOW 
  }
);

// Retrieve relevant memories
const memories = await client.retrieve(
  'What files have been edited recently?',
  { limit: 10 }
);

client.close();
```

## MembraneClient

### Constructor

```typescript
new MembraneClient(addr?: string, options?: MembraneClientOptions)
```

Create a new client instance.

<ParamField path="addr" type="string" default="localhost:9090">
  gRPC server address in `host:port` format.
</ParamField>

<ParamField path="options" type="MembraneClientOptions">
  Optional configuration object.
</ParamField>

#### MembraneClientOptions

<ParamField path="tls" type="boolean" default={false}>
  Enable TLS transport.
</ParamField>

<ParamField path="tlsCaCertPath" type="string">
  Path to PEM-encoded CA certificate for server verification.
</ParamField>

<ParamField path="apiKey" type="string">
  Optional Bearer token for server authentication.
</ParamField>

<ParamField path="timeoutMs" type="number">
  Default timeout in milliseconds for all RPC calls.
</ParamField>

<ParamField path="transport" type="RpcTransport">
  Custom transport implementation (advanced use only).
</ParamField>

### Example: Secure Connection

```typescript
const client = new MembraneClient(
  'membrane.example.com:443',
  {
    tls: true,
    apiKey: 'your-api-key',
    timeoutMs: 10000
  }
);
```

## Ingestion Methods

### ingestEvent

```typescript
ingestEvent(
  eventKind: string,
  ref: string,
  options?: IngestEventOptions
): Promise<MemoryRecord>
```

Ingest an event into the memory substrate.

<ParamField path="eventKind" type="string" required>
  Type of event (e.g., `"file_edit"`, `"error"`, `"user_action"`).
</ParamField>

<ParamField path="ref" type="string" required>
  Reference identifier for the event source.
</ParamField>

<ParamField path="options.summary" type="string">
  Human-readable summary of the event.
</ParamField>

<ParamField path="options.sensitivity" type="Sensitivity | string" default="low">
  Sensitivity classification: `"public"`, `"low"`, `"medium"`, `"high"`, or `"hyper"`.
</ParamField>

<ParamField path="options.source" type="string" default="typescript-client">
  Source identifier for provenance tracking.
</ParamField>

<ParamField path="options.tags" type="string[]">
  Optional tags for categorization.
</ParamField>

<ParamField path="options.scope" type="string">
  Visibility scope for the record.
</ParamField>

<ParamField path="options.timestamp" type="string">
  RFC 3339 timestamp; defaults to current time.
</ParamField>

#### Example

```typescript
const record = await client.ingestEvent(
  'deployment',
  'prod-v2.3.1',
  {
    summary: 'Deployed version 2.3.1 to production',
    sensitivity: Sensitivity.MEDIUM,
    tags: ['production', 'deployment'],
    scope: 'engineering'
  }
);
```

## Retrieval Methods

### retrieve

```typescript
retrieve(
  taskDescriptor: string,
  options?: RetrieveOptions
): Promise<MemoryRecord[]>
```

Retrieve memories relevant to a task descriptor.

<ParamField path="taskDescriptor" type="string" required>
  Natural-language description of the current task.
</ParamField>

<ParamField path="options.trust" type="TrustContext">
  Trust context controlling access permissions. Defaults to minimal context with `Sensitivity.LOW`.
</ParamField>

<ParamField path="options.memoryTypes" type="Array<MemoryType | string>">
  Filter for specific memory types: `"episodic"`, `"working"`, `"semantic"`, `"competence"`, or `"plan_graph"`.
</ParamField>

<ParamField path="options.minSalience" type="number" default={0}>
  Minimum salience threshold (0.0 to 1.0).
</ParamField>

<ParamField path="options.limit" type="number" default={10}>
  Maximum number of records to return.
</ParamField>

#### Example

```typescript
import { MemoryType, createDefaultTrustContext } from '@gustycube/membrane';

const trust = createDefaultTrustContext();
trust.max_sensitivity = Sensitivity.MEDIUM;
trust.authenticated = true;
trust.actor_id = 'agent-1';

const memories = await client.retrieve(
  'Fix the authentication bug in the login flow',
  {
    trust,
    memoryTypes: [MemoryType.EPISODIC, MemoryType.SEMANTIC],
    minSalience: 0.3,
    limit: 5
  }
);
```

### retrieveWithSelection

```typescript
retrieveWithSelection(
  taskDescriptor: string,
  options?: RetrieveOptions
): Promise<RetrieveResult>
```

Retrieve memories with additional selection metadata.

<ParamField path="taskDescriptor" type="string" required>
  Natural-language description of the current task.
</ParamField>

<ParamField path="options" type="RetrieveOptions">
  Same options as `retrieve()`.
</ParamField>

Returns a `RetrieveResult` object containing:
- `records`: Array of `MemoryRecord` instances
- `selection`: Optional `SelectionResult` with confidence metrics

#### Example

```typescript
const result = await client.retrieveWithSelection(
  'Find recent deployment errors',
  { limit: 10 }
);

console.log(`Found ${result.records.length} records`);
if (result.selection) {
  console.log(`Selection confidence: ${result.selection.confidence}`);
}
```

## Type Definitions

### MemoryRecord

The atomic unit of storage in Membrane.

```typescript
interface MemoryRecord {
  id: string;
  type: MemoryType | string;
  sensitivity: Sensitivity | string;
  confidence: number;
  salience: number;
  scope?: string;
  tags?: string[];
  created_at?: string;
  updated_at?: string;
  lifecycle?: Lifecycle;
  provenance?: Provenance;
  relations?: Relation[];
  payload?: unknown;
  audit_log?: AuditEntry[];
}
```

### TrustContext

Controls access permissions during retrieval.

```typescript
interface TrustContext {
  max_sensitivity: Sensitivity | string;
  authenticated: boolean;
  actor_id: string;
  scopes: string[];
}
```

Create a default trust context:

```typescript
import { createDefaultTrustContext } from '@gustycube/membrane';

const trust = createDefaultTrustContext();
// Returns: { max_sensitivity: 'low', authenticated: false, actor_id: '', scopes: [] }
```

## Error Handling

The client throws `MembraneError` for gRPC errors:

```typescript
import { MembraneError } from '@gustycube/membrane';

try {
  await client.ingestEvent('test', 'ref');
} catch (err) {
  if (err instanceof MembraneError) {
    console.error('Membrane error:', err.message);
    console.error('Status code:', err.code);
  }
}
```
