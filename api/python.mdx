## Installation

Install the Membrane Python client using pip:

```bash
pip install membrane-client
```

Requires Python 3.10 or higher.

## Quick Start

```python
from membrane import MembraneClient, Sensitivity

client = MembraneClient("localhost:9090")

# Ingest an event
record = client.ingest_event(
    event_kind="file_edit",
    ref="src/main.py",
    summary="Updated main entry point",
    sensitivity=Sensitivity.LOW
)

# Retrieve relevant memories
memories = client.retrieve(
    "What files have been edited recently?",
    limit=10
)

client.close()
```

## MembraneClient

### Constructor

```python
MembraneClient(
    addr: str = "localhost:9090",
    *,
    tls: bool = False,
    tls_ca_cert: str | None = None,
    api_key: str | None = None,
    timeout: float | None = None
)
```

Create a new client instance.

<ParamField path="addr" type="str" default="localhost:9090">
  gRPC server address in `host:port` format.
</ParamField>

<ParamField path="tls" type="bool" default={false}>
  Enable TLS transport. When `True` without `tls_ca_cert`, uses system root certificates.
</ParamField>

<ParamField path="tls_ca_cert" type="str">
  Path to PEM-encoded CA certificate file for server verification. Implies `tls=True`.
</ParamField>

<ParamField path="api_key" type="str">
  Optional Bearer token for server authentication.
</ParamField>

<ParamField path="timeout" type="float">
  Default timeout in seconds for all RPC calls. `None` means no timeout.
</ParamField>

### Example: Secure Connection

```python
client = MembraneClient(
    "membrane.example.com:443",
    tls=True,
    api_key="your-api-key",
    timeout=10.0
)
```

### Context Manager Support

The client supports the context manager protocol:

```python
with MembraneClient("localhost:9090") as client:
    record = client.ingest_event(
        event_kind="test",
        ref="example"
    )
    # Client automatically closed on exit
```

## Ingestion Methods

### ingest_event

```python
ingest_event(
    event_kind: str,
    ref: str,
    *,
    summary: str = "",
    sensitivity: Sensitivity | str = Sensitivity.LOW,
    source: str = "python-client",
    tags: Sequence[str] | None = None,
    scope: str = "",
    timestamp: str | None = None
) -> MemoryRecord
```

Ingest an event into the memory substrate.

<ParamField path="event_kind" type="str" required>
  Type of event (e.g., `"file_edit"`, `"error"`, `"user_action"`).
</ParamField>

<ParamField path="ref" type="str" required>
  Reference identifier for the event source.
</ParamField>

<ParamField path="summary" type="str" default="">
  Human-readable summary of the event.
</ParamField>

<ParamField path="sensitivity" type="Sensitivity | str" default="Sensitivity.LOW">
  Sensitivity classification: `Sensitivity.PUBLIC`, `LOW`, `MEDIUM`, `HIGH`, or `HYPER`.
</ParamField>

<ParamField path="source" type="str" default="python-client">
  Source identifier for provenance tracking.
</ParamField>

<ParamField path="tags" type="Sequence[str]">
  Optional tags for categorization.
</ParamField>

<ParamField path="scope" type="str" default="">
  Visibility scope for the record.
</ParamField>

<ParamField path="timestamp" type="str">
  RFC 3339 timestamp; defaults to current time.
</ParamField>

#### Example

```python
from membrane import Sensitivity

record = client.ingest_event(
    event_kind="deployment",
    ref="prod-v2.3.1",
    summary="Deployed version 2.3.1 to production",
    sensitivity=Sensitivity.MEDIUM,
    tags=["production", "deployment"],
    scope="engineering"
)
```

## Retrieval Methods

### retrieve

```python
retrieve(
    task_descriptor: str,
    *,
    trust: TrustContext | None = None,
    memory_types: Sequence[MemoryType | str] | None = None,
    min_salience: float = 0.0,
    limit: int = 10
) -> list[MemoryRecord]
```

Retrieve memories relevant to a task descriptor.

<ParamField path="task_descriptor" type="str" required>
  Natural-language description of the current task.
</ParamField>

<ParamField path="trust" type="TrustContext">
  Trust context controlling access permissions. Defaults to minimal context with `Sensitivity.LOW`.
</ParamField>

<ParamField path="memory_types" type="Sequence[MemoryType | str]">
  Filter for specific memory types: `MemoryType.EPISODIC`, `WORKING`, `SEMANTIC`, `COMPETENCE`, or `PLAN_GRAPH`.
</ParamField>

<ParamField path="min_salience" type="float" default={0.0}>
  Minimum salience threshold (0.0 to 1.0).
</ParamField>

<ParamField path="limit" type="int" default={10}>
  Maximum number of records to return.
</ParamField>

#### Example

```python
from membrane import TrustContext, MemoryType, Sensitivity

trust = TrustContext(
    max_sensitivity=Sensitivity.MEDIUM,
    authenticated=True,
    actor_id="agent-1"
)

memories = client.retrieve(
    "Fix the authentication bug in the login flow",
    trust=trust,
    memory_types=[MemoryType.EPISODIC, MemoryType.SEMANTIC],
    min_salience=0.3,
    limit=5
)
```

### retrieve_with_selection

```python
retrieve_with_selection(
    task_descriptor: str,
    *,
    trust: TrustContext | None = None,
    memory_types: Sequence[MemoryType | str] | None = None,
    min_salience: float = 0.0,
    limit: int = 10
) -> RetrieveResult
```

Retrieve memories with additional selection metadata.

<ParamField path="task_descriptor" type="str" required>
  Natural-language description of the current task.
</ParamField>

<ParamField path="trust" type="TrustContext">
  Trust context controlling access.
</ParamField>

<ParamField path="memory_types" type="Sequence[MemoryType | str]">
  Filter for specific memory types.
</ParamField>

<ParamField path="min_salience" type="float" default={0.0}>
  Minimum salience threshold.
</ParamField>

<ParamField path="limit" type="int" default={10}>
  Maximum number of records.
</ParamField>

Returns a `RetrieveResult` object containing:
- `records`: List of `MemoryRecord` instances
- `selection`: Optional `SelectionResult` with confidence metrics

#### Example

```python
result = client.retrieve_with_selection(
    "Find recent deployment errors",
    limit=10
)

print(f"Found {len(result.records)} records")
if result.selection:
    print(f"Selection confidence: {result.selection.confidence}")
```

## Type Definitions

### MemoryRecord

The atomic unit of storage in Membrane.

```python
@dataclass
class MemoryRecord:
    id: str
    type: MemoryType
    sensitivity: Sensitivity
    confidence: float
    salience: float
    scope: str
    tags: list[str]
    created_at: str
    updated_at: str
    lifecycle: Lifecycle | None
    provenance: Provenance | None
    relations: list[Relation]
    payload: Any
    audit_log: list[AuditEntry]
```

The `MemoryRecord` class provides:
- `from_dict(data: dict[str, Any]) -> MemoryRecord`: Construct from JSON
- `to_dict() -> dict[str, Any]`: Serialize to JSON

## Complete Example

```python
from membrane import (
    MembraneClient,
    MemoryType,
    OutcomeStatus,
    Sensitivity,
    TrustContext,
)

# Create client with secure connection
with MembraneClient(
    "membrane.example.com:443",
    tls=True,
    api_key="your-api-key"
) as client:
    # Ingest an event
    event_record = client.ingest_event(
        event_kind="deployment",
        ref="prod-v3.0.0",
        summary="Deployed new authentication system",
        sensitivity=Sensitivity.MEDIUM,
        tags=["production", "auth"]
    )
    
    # Ingest tool output
    tool_record = client.ingest_tool_output(
        tool_name="run_tests",
        args={"suite": "integration"},
        result={"passed": 42, "failed": 0},
        depends_on=[event_record.id]
    )
    
    # Record outcome
    client.ingest_outcome(
        target_record_id=event_record.id,
        outcome_status=OutcomeStatus.SUCCESS
    )
    
    # Retrieve relevant memories
    trust = TrustContext(
        max_sensitivity=Sensitivity.MEDIUM,
        authenticated=True,
        actor_id="admin-1"
    )
    
    memories = client.retrieve(
        "What happened with the authentication deployment?",
        trust=trust,
        memory_types=[MemoryType.EPISODIC],
        limit=10
    )
    
    for memory in memories:
        print(f"[{memory.type.value}] {memory.id}")
        print(f"  Salience: {memory.salience}")
        print(f"  Payload: {memory.payload}")
    
    # Reinforce important memories
    client.reinforce(
        record_id=event_record.id,
        actor="admin-1",
        rationale="Critical deployment milestone"
    )
    
    # Get metrics
    metrics = client.get_metrics()
    print(f"Total records: {metrics.get('total_records', 'N/A')}")
```
