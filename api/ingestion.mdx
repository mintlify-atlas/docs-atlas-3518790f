## Overview

Ingestion methods allow you to store different types of memory records into the Membrane system. All ingestion methods return an `IngestResponse` containing the created memory record.

## IngestEvent

Record significant events that occur in your system.

### Request Parameters

<ParamField path="source" type="string" required>
  Identifier of the system or agent creating the event
</ParamField>

<ParamField path="event_kind" type="string" required>
  Type or category of the event
</ParamField>

<ParamField path="ref" type="string" required>
  Reference or identifier for the event
</ParamField>

<ParamField path="summary" type="string" required>
  Human-readable description of the event
</ParamField>

<ParamField path="timestamp" type="string" required>
  RFC 3339 formatted timestamp
</ParamField>

<ParamField path="tags" type="string[]">
  Array of tags for categorization
</ParamField>

<ParamField path="scope" type="string">
  Visibility scope for the event
</ParamField>

<ParamField path="sensitivity" type="string">
  Sensitivity level (e.g., "public", "private", "confidential")
</ParamField>

### Response

<ResponseField name="record" type="bytes" required>
  JSON-encoded MemoryRecord containing the stored event
</ResponseField>

### Example

<Tabs>
  <Tab title="Go">
    ```go
    req := &membranev1.IngestEventRequest{
        Source:      "agent-1",
        EventKind:   "user_action",
        Ref:         "action-123",
        Summary:     "User completed onboarding",
        Timestamp:   time.Now().Format(time.RFC3339),
        Tags:        []string{"onboarding", "milestone"},
        Scope:       "team",
        Sensitivity: "internal",
    }
    
    resp, err := client.IngestEvent(ctx, req)
    if err != nil {
        log.Fatal(err)
    }
    
    // Parse the memory record
    var record MemoryRecord
    json.Unmarshal(resp.Record, &record)
    ```
  </Tab>
  
  <Tab title="Python">
    ```python
    import json
    from datetime import datetime
    from membrane.v1 import membrane_pb2
    
    req = membrane_pb2.IngestEventRequest(
        source="agent-1",
        event_kind="user_action",
        ref="action-123",
        summary="User completed onboarding",
        timestamp=datetime.now().isoformat(),
        tags=["onboarding", "milestone"],
        scope="team",
        sensitivity="internal"
    )
    
    resp = stub.IngestEvent(req)
    record = json.loads(resp.record)
    ```
  </Tab>
  
  <Tab title="TypeScript">
    ```typescript
    const req: IngestEventRequest = {
      source: 'agent-1',
      eventKind: 'user_action',
      ref: 'action-123',
      summary: 'User completed onboarding',
      timestamp: new Date().toISOString(),
      tags: ['onboarding', 'milestone'],
      scope: 'team',
      sensitivity: 'internal',
    };
    
    client.ingestEvent(req, (err, resp) => {
      if (err) throw err;
      const record = JSON.parse(resp.record.toString());
    });
    ```
  </Tab>
</Tabs>

## IngestToolOutput

Store the results of tool executions, including arguments and dependencies.

### Request Parameters

<ParamField path="source" type="string" required>
  Identifier of the agent or system executing the tool
</ParamField>

<ParamField path="tool_name" type="string" required>
  Name of the tool that was executed
</ParamField>

<ParamField path="args" type="bytes" required>
  JSON-encoded map of tool arguments
</ParamField>

<ParamField path="result" type="bytes" required>
  JSON-encoded result value from the tool
</ParamField>

<ParamField path="depends_on" type="string[]">
  Array of record IDs this tool output depends on
</ParamField>

<ParamField path="timestamp" type="string" required>
  RFC 3339 formatted timestamp
</ParamField>

<ParamField path="tags" type="string[]">
  Array of tags for categorization
</ParamField>

<ParamField path="scope" type="string">
  Visibility scope
</ParamField>

<ParamField path="sensitivity" type="string">
  Sensitivity level
</ParamField>

### Response

<ResponseField name="record" type="bytes" required>
  JSON-encoded MemoryRecord containing the stored tool output
</ResponseField>

### Example

<Tabs>
  <Tab title="Go">
    ```go
    args, _ := json.Marshal(map[string]interface{}{
        "query": "weather in London",
        "units": "celsius",
    })
    
    result, _ := json.Marshal(map[string]interface{}{
        "temperature": 15,
        "condition": "cloudy",
    })
    
    req := &membranev1.IngestToolOutputRequest{
        Source:    "agent-1",
        ToolName:  "weather_api",
        Args:      args,
        Result:    result,
        DependsOn: []string{"rec-456"},
        Timestamp: time.Now().Format(time.RFC3339),
        Tags:      []string{"tool", "api_call"},
    }
    
    resp, err := client.IngestToolOutput(ctx, req)
    ```
  </Tab>
  
  <Tab title="Python">
    ```python
    import json
    from datetime import datetime
    
    args = json.dumps({"query": "weather in London", "units": "celsius"})
    result = json.dumps({"temperature": 15, "condition": "cloudy"})
    
    req = membrane_pb2.IngestToolOutputRequest(
        source="agent-1",
        tool_name="weather_api",
        args=args.encode(),
        result=result.encode(),
        depends_on=["rec-456"],
        timestamp=datetime.now().isoformat(),
        tags=["tool", "api_call"]
    )
    
    resp = stub.IngestToolOutput(req)
    ```
  </Tab>
</Tabs>

## Common Response Structure

All ingestion methods return an `IngestResponse` with the following structure:

```json
{
  "record": "<JSON-encoded MemoryRecord>"
}
```

The `record` field contains a JSON-encoded `MemoryRecord` with properties like:
- `id`: Unique identifier for the record
- `type`: Type of memory record
- `content`: The actual content stored
- `metadata`: Additional metadata
- `salience`: Initial salience score
- `created_at`: Creation timestamp
- `provenance`: Source and lineage information
