## Using the Go Library

<Steps>
  <Step title="Install Membrane">
    Add Membrane to your Go project:

    ```bash
    go get github.com/GustyCube/membrane
    ```
  </Step>

  <Step title="Initialize and Start">
    Create a Membrane instance with default configuration:

    ```go
    package main

    import (
        "context"
        "log"

        "github.com/GustyCube/membrane/pkg/membrane"
    )

    func main() {
        cfg := membrane.DefaultConfig()
        cfg.DBPath = "my-agent.db"

        m, err := membrane.New(cfg)
        if err != nil {
            log.Fatal(err)
        }
        defer m.Stop()

        ctx := context.Background()
        m.Start(ctx)
        
        // Your agent code here
    }
    ```
  </Step>

  <Step title="Ingest Your First Memory">
    Record an episodic event:

    ```go
    import "github.com/GustyCube/membrane/pkg/ingestion"

    rec, err := m.IngestEvent(ctx, ingestion.IngestEventRequest{
        Source:    "my-agent",
        EventKind: "tool_call",
        Ref:       "task#1",
        Summary:   "Executed database migration successfully",
        Tags:      []string{"db", "migration"},
    })
    if err != nil {
        log.Fatal(err)
    }
    log.Printf("Created record: %s", rec.ID)
    ```
  </Step>

  <Step title="Retrieve Memory">
    Query memory with trust context:

    ```go
    import (
        "github.com/GustyCube/membrane/pkg/retrieval"
        "github.com/GustyCube/membrane/pkg/schema"
    )

    resp, err := m.Retrieve(ctx, &retrieval.RetrieveRequest{
        TaskDescriptor: "database operations",
        Trust: &retrieval.TrustContext{
            MaxSensitivity: schema.SensitivityMedium,
            Authenticated:  true,
        },
        MemoryTypes: []schema.MemoryType{
            schema.MemoryTypeCompetence,
            schema.MemoryTypeSemantic,
        },
        Limit: 10,
    })
    if err != nil {
        log.Fatal(err)
    }

    for _, r := range resp.Records {
        log.Printf("Found: %s (type=%s, confidence=%.2f)", 
            r.ID, r.Type, r.Confidence)
    }
    ```
  </Step>
</Steps>

## Running the Daemon

<Steps>
  <Step title="Clone and Build">
    ```bash
    git clone https://github.com/GustyCube/membrane.git
    cd membrane
    make build
    ```
  </Step>

  <Step title="Start the Daemon">
    ```bash
    # Start with default SQLite storage
    ./bin/membraned

    # Or with custom configuration
    ./bin/membraned --config config.yaml --addr :9090
    ```

    <Note>By default, Membrane listens on `:9090` and stores data in `membrane.db`</Note>
  </Step>

  <Step title="Connect a Client">
    <Tabs>
      <Tab title="TypeScript">
        ```bash
        npm install @gustycube/membrane
        ```

        ```typescript
        import { MembraneClient, Sensitivity } from "@gustycube/membrane";

        const client = new MembraneClient("localhost:9090");

        const record = await client.ingestEvent("tool_call", "task#1", {
          summary: "Ran database migration successfully",
          tags: ["db", "migration"]
        });

        const results = await client.retrieve("database operations", {
          trust: {
            max_sensitivity: Sensitivity.MEDIUM,
            authenticated: true
          }
        });

        console.log(`Found ${results.length} records`);
        client.close();
        ```
      </Tab>

      <Tab title="Python">
        ```bash
        pip install membrane-client
        ```

        ```python
        from membrane import MembraneClient, Sensitivity, TrustContext

        client = MembraneClient("localhost:9090")

        record = client.ingest_event(
            source="my-agent",
            event_kind="tool_call",
            ref="task#1",
            summary="Ran database migration successfully",
            tags=["db", "migration"]
        )

        results = client.retrieve(
            task_descriptor="database operations",
            trust=TrustContext(
                max_sensitivity=Sensitivity.MEDIUM,
                authenticated=True
            )
        )

        print(f"Found {len(results)} records")
        client.close()
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
